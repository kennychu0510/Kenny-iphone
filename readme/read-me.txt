# homepage
My homepage for mobile

Best viewed on mobile. Tested on iphone 12. This website layout is inspired by the IOS UX/UI. It is written in pure HTML, CSS and Javascript, no frameworks are used. It contains clickable app icons which direct viewers to the corresponding websites.
